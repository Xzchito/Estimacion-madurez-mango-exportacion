# Estimación de madurez de mango para exportación

## Descripción General del Proyecto
Este proyecto tiene como objetivo predecir la vida útil del mango mediante el análisis del viraje de color en su cáscara. El sistema procesa imágenes digitales capturadas en un entorno con iluminación y fondo controlados para garantizar una segmentación precisa utilizando el espacio de color HSV. A través del análisis de histogramas en el canal Hue (Matiz), se evalúa la distribución de píxeles y se realiza una cuantificación estadística que permite estimar de manera automatizada el estado de madurez del fruto para su posterior exportación.
